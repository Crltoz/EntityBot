# EntityBot

This bot was born with the idea of providing Dead By Daylight players with their statistics without the need to search for them on the web. Therefore, this Discord bot was created with simple commands to retrieve statistics by making API requests to the Dead By Daylight statistics [website](https://dbd.tricky.lol) created by Tricky.
